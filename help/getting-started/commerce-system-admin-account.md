# Update System Administrator access from Adobe Admin Console

Only a current System Administrator can assign, change, or remove the System Administrator role in Adobe Admin Console for Adobe products and services other than Adobe Commerce.

To assign a new System Administrator or change an existing admin role for other Adobe products and services, including App Builder, an existing administrator must sign in to the Adobe Admin Console and update the organization's administrative roles. See [Administrator roles](https://experienceleague.adobe.com/en/docs/support-resources/adobe-support-tools-guide/adobe-admin-console/admin-roles).

## Assign a new System Administrator

1. Sign in to the Adobe Admin Console.
1. Go to **[!UICONTROL Users]** > **[!UICONTROL Administrators]**.
1. Select **[!UICONTROL Add Admin]**.
1. Enter the user's name or email address.
1. Select **[!UICONTROL System Administrator]**.
1. Select **[!UICONTROL Save]**.

## Edit or remove an existing administrator

1. Sign in to the Adobe Admin Console.
1. Go to **[!UICONTROL Users]** > **[!UICONTROL Administrators]**.
1. Select the administrator you want to update.
1. In **[!UICONTROL Administrative Rights]**, select **[!UICONTROL Edit admin rights]**.
1. Update or remove the assigned admin role, then select **[!UICONTROL Save]**.
