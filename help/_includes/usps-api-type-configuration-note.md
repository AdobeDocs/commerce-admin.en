# USPS API type configuration options

>[!NOTE]
>
>These instructions include steps to select the USPS API type for integration: Web Tools API or REST APIs. USPS no longer supports the Web Tools API, so you must configure their REST APIs. If the REST APIs option is not displayed, you must apply the [USPS REST API Migration quality patch](https://experienceleague.adobe.com/en/docs/commerce-operations/tools/quality-patches-tool/patches-available-in-qpt/v1-1-70/ac-15210)(AC-1520) to your Commerce application.
